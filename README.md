# 3rd-Yr-Week-3-Module
## Week 3 Programming Module
This repository contains Java programs analyzing different data sets:
- EmergencyResponseAnalysis.java: Calculates average response times.
- OrderProcessingAnalysis.java: Evaluates order efficiency.
- TournamentScores.java: Computes player rankings.
- TradeValueAnalysis.java: Assesses trade performance metrics.
